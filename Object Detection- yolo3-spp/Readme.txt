
YOLO3- Object Detection:
------------------------
1. Download Pre trained Weights, configuration files and class names
	i. yolov3-spp.weights, 
	ii. yolov3-spp.cfg,
	iii. coco.names
	We can download weights and configuration files from following link
	https://pjreddie.com/darknet/yolo/

2. Run object detection_YOLO_spp.py 
	This will read image, video and also webcam live
	It will ask that select 1 for Image Detection, 2 for Video, 3 for Webcam
	Next it will ask enter file name you need to give you path of image or video
	it will detect object and saves the file.
